    Edição Básica de Arquivos

    vi/vim

    • vi --version - mostra se é o VI ou o VIM
    • vi arquivo.txt
    • 3 modos:
      • ESC - modo de navegação
      • I - modo de inserção
      • : - como de comandos
    • /Linux - procura pela primeira ocorrêncida da palavra
      • n - próxima ocorrência
      • shift + N - próxima ocorrência de baixo para cima
    • ?Linux - pesquisa da onde estou para cima
    • i - começa a editar da linha que estou
    • o (minúsculo) - começa da linha de baixo
    • O (maiúsculo) - começa da linha de cima
    • a (minúsculo) - começa a editar do próximo caractere
    • A (maiúsculo) - começa a editar do final da linha
    • cc - recorta a linha e entra no modo de edição
    • p - cola a linha recortada
    • dd - recorta e não entra no modo de edição
    • d3d - recorta as 3 linhas seguintes
    • yy - copia a linha
    • y3y - copia 3 linhas
    • :w - salvar o arquivo
    • :q - sair do arquivo
    • :q! - sair sem salvar
    • :wq - sair e salvar
    • :x - sair e salvar
    • :wq! - sair e salvar forçado
    • ZZ (maiúsculo) - sair e salvar
    • :!ls /tmp - executa o comando
    • vi - acessa o vi sem arquivo definido
      • :wq arquivo-novo.txt
    • :e - atualiza o arquivo caso outro processo tenha alterado ele, sobrescreve sua edição caso tenha feito algo

    nano e emacs

    • nano texto.txt
    • ctrl + g - manual do nano
    • ctrl + o - salva o arquivo
    • ctrl + x - sair do arquivo
    • end - começo da linha
    • home - começo da linha
    • alt + / - final do texto
    • alt + \ - começo do arquivo
    • ctrl + v - próxima página
    • ctrl + p - página anterior
    • ctrl + _ - ir para certa linha
    • ctrl + w - pesquisar palavra
    • emacs texto.txt
    • export DISPLAY=
      • abrir o emacs via texto mesmo com interface gráfica
    • ctrl + x + ctrl + c - sai do emacs
    • emacs - abre sem arquivo
      • ctrl + x + ctrl + f - procura ou cria um novo arquivo
    • ctrl + x + ctrl + s - salva o arquivo
    • ctrl + x + s - pergunta antes de salvar
    • ctrl + x + ctrl + w - salvar como
    • ctrl + k - recorta a partir do cursor
      • pode fazer em várias linhas, guarda em buffer
    • ctrl + y - cola
    • ctrl + e - fim da linha
    • ctrl + a - começo da linha
    • alt + shift + > - fim do texto
    • alt + shift + < - começo do texto
    • ctrl + / - desfaz a última ação - undo
    • alt + x - define o modo de operação conforme a linguagem ou tipo de texto

    Definindo o Editor padrão

    • select-editor - seleciona o editor padrão
    • editor - abre o editor padrão (não o select-editor)
    • update-alternatives --config editor - selecionar o editor
    • echo $EDITOR - variável de ambiente do editor padrão
    • export EDITOR=nano
    • sobrescreve o select-editor
