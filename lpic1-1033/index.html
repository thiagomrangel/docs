    103.3 - Gerenciamento Básico de Arquivos

    cd, ls, file

    cd

    • cd - - volta para o diretório que eu estava
    • cd ~ - volta para o home do meu usuário
    • pwd - diretório que estou
    • cd ../.. - volta 2 diretórios

    ls

    • ls -a - mostra diretórios ocultos
    • ls -l - lista detalhada dos arquivos
      • . e .. no começo - diretório atual e anterior
      • ls -h - facilita leitura para humanos
      • ls -R - recursivo, mostra o conteúdo de todos os diretórios
      • ls Aula1* - mostra tudo que começa com a sintaxe
      • ls Aula1? - ? substitui o caractere
      • ls Aula[123] - mostra aulas 1, 2 e 3
      • ls Aula[123] - nega a lista
      • ls Aula[1-5] - de 1 a 5
      • ls Aula[1-5]?
      • ls Aula{10,20,30}
      • ls A{ula,ULA}1

    file

    • file alunos.txt - mostra o tipo do arquivo

    touch, cp, mv

    cp

    • cp -opcoes origem destino
    • cp -v alunos* /tmp/ - verbose, mostra todos os passos do processo
    • cp -i - pergunta se vai sobrescresver um arquivo
    • cp -r - recursivo, copiar diretórios
    • cp -p - preservar as características do arquivo
    • cp A B - copia o arquivo com um novo nome

    mv

    • mv teste /tmp - move o arquivo
    • mv A B - renomeia o arquivo

    touch

    • touch linux.txt - cria o arquivo vazio
    • ls -l - mostra a data da última alteração
    • touch arquivo - atualiza a hora da última modificação do arquivo
    • touch -a arquivo - altera a data do último acesso
    • touch -m arquivo - altera a data da última modificação
    • touch -t ANOMESDIAHORA - altera a data da última alteração

    rm, mkdir, rmdir, find

    rm

    • rm arquivo - remove o arquivo
    • rm -i arquivo - pede confirmação
    • rm -v arquivo - mostra detalhes da remoção
    • rm -r diretório - recursivo, remove diretórios
    • rmdir diretório - remóve diretórios VAZIOS
    • rmdir -p linux/ubuntu - apaga a estrutura inteira
    • mkdir diretório - cria o diretório
    • mkdir linux/ubuntu - cria um diretório dentro do outro
    • mkdir -p (--parents) curso/lpi/versao1 - cria toda a estrutura de diretórios
    • find /home -name aula - procurar no /home o arquivo
    • find /home -user lpi2 -name aula
    • find ./ -ctime -1 - mostra os arquivos a partir do diretório que eu estou que foram executados no máximo 1 dia antes
    • -d - tipo de arquivo diretório
    • -not nega o parâmetro seguinte, não deve existir
    • mtime - tempo de modificação
    • find /var/log/ -user ricardo -exec cp -r {} /home/lpi/logs_ricardo/ \;

    tar, gzip, bzip2, xz

    tar

    • tar cpvf backup.tar novo* - só agrupa, não compacta
    • c - criar
    • f - definir o nome
    • p - preservar preservar as permissões
    • v - verbose
    • t - listar
    • z - gzip
    • j - gzip2
    • J - xz
    • x - descompactar

    gzip

    • gzip backup.tar - remove o arquivo original e compacta
    • k - manter o original
    • gunzip backup.tar.gz - remove o arquivo compactado e descompacta

    bzip2

    • bzip2 -k backup.tar
    • d - descompactar
    • bunzip backup.tar.bz2

    xz

    • xz -k arquivo.tar
    • unxz ou xz -d

    file

    • file arquivo.tgz - mostra as propriedades da compactação

    cpio, dd

    cpio

    • agrupa uma lista de arquivos
    • find ./ -name "novo*" | cpio -o > backup.cpio
    • o - cria o arquivo
    • find ./ -name "novo*" | cpio -o | gzip > backup.cpio.gz
    • compacta
    • cpio -i < backup.cpio
    • desagrupa o arquivo
    • d - cria a árvore de diretórios
    • gunzip -c backup.cpio.gz | cpio -i
    • descompactar o arquivo ## dd
    • copiar uma partição inteira
      • dd if=/dev/sr0 of=image.img
      • cria um arquivo de imagem, copia byte a byte
